# Legacy — the original Windows version

`Fast Debate Paste 1.6.ahk` is the original AutoHotkey v2 script this Mac
app ports. It is kept here for reference only: the Mac app is the
maintained version, and the script's Zotero / Research Tracker /
copy-header / copy-URL actions were deliberately dropped from the port.

To use it on Windows you need [AutoHotkey v2](https://www.autohotkey.com/);
run the script and it registers the same Ctrl+Shift hotkey family.
